    MsBuildProjectTargets

    This project will assist in automating deployment process for ASP.Net projects

    Objective of the project

    The objective is to create MsBuild targets which will help in automating the deployment process for .NET projects. A typical product deployment process will en compose the following steps.

    1. Running the test projects successfully
    2. Assigning a proper version to the project
    3. Publishing the project
    4. Packaging the product

    Prerequisites

    1. .NET Framework 3.5 or above
    2. MsBuild version 4.0.6
    3. MsTest package
